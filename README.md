ArtMatch - Backend & Mobile App

ArtMatch est une plateforme de mise en relation pour artistes. Ce projet démontre mes capacités à concevoir une architecture **Backend scalable** et à la connecter à une interface mobile.

 *Stack Technique
- **Backend :** Node.js, Express.js
- **Base de données :** PostgreSQL via **Supabase**
- **Mobile :** React Native avec Expo
- **Gestion d'état :** Hooks React & Context API

*Architecture Backend
- Modélisation de données PostgreSQL (Utilisateurs, Matchs, Profils artistiques).
- API RESTful pour la gestion des authentifications et des flux de données.
- Intégration de Supabase pour la perséance des données et le stockage d'objets.
